# 简介

本组件库的组件均提炼自智慧社区项目的开发过程中，致力于加快日常开发。

联系：skype。

提示

组件库可能未经完整测试，所以可能会存在bug。

本项目所有代码均为es6以上语法，需要在vue.config.js文件里配置进行转义：

{
    transpileDependencies: [
        'dolphin-plugin-tools',
        /@hui-pro/,
        /img-upload/,
        /ol-measure/
        // ...
    ],
}

本项目仓库：http://iris.hikvision.com.cn/wanglin25/vue_components。
