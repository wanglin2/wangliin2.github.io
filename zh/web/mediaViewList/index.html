# MediaViewList 媒体预览组件

支持图片、视频、音频的显示及预览。

图片可点击进行放大预览，音频可控制播放。

视频不支持显示预览图，视频点击后是打开浏览器窗口进行播放。

视频和音频的样式较丑，可自行覆盖修改。

# 安装

$ npm i mediaviewlist

# 引入

import MediaViewList from 'mediaviewlist'
Vue.use(MediaViewList)
