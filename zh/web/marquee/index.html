# Marquee 跑马灯组件

该组件同时适用于PC端和APP端。

# 安装

$ npm i i-marquee

# 引入

import Marquee from 'i-marquee'
Vue.use(Marquee)

# 基础用法

没有超出不会滚动
我很长很长很长很长很长很长很长
我很快很快很快很快很快很快很快
我走走停停走走停停走走停停走走停停走走停停
基础用法
<template>
  <div class="container">
    <div class="box">
      <Marquee :offset="10">没有超出不会滚动</Marquee>
    </div>
    <div class="box">
      <Marquee :offset="10" :delay="1000" :loop="3">我很长很长很长很长很长很长很长</Marquee>
    </div>
    <div class="box">
      <Marquee :offset="10" :delay="1000" :loop="3" :speed="10">我很快很快很快很快很快很快很快</Marquee>
    </div>
    <div class="box">
      <Marquee ref="Marquee" :offset="10" :delay="1000" :loop="10" :speed="20" :auto="false">我走走停停走走停停走走停停走走停停走走停停</Marquee>
    </div>
  </div>
</template>

<script>
export default {
  data () {
    return {}
  },
  mounted () {
    setTimeout(() => {
        this.$refs.Marquee.start();
        // 滚动中调用start方法无效
        setTimeout(() => {
            this.$refs.Marquee.start();
        }, 1000);
        // 暂停
        setTimeout(() => {
            this.$refs.Marquee.stop();
        }, 3000);
        // 继续滚动
        setTimeout(() => {
            this.$refs.Marquee.start();
        }, 5000);
        // 复位
        setTimeout(() => {
            this.$refs.Marquee.reset();
        }, 7000);
        // 重新开始滚动
        setTimeout(() => {
            this.$refs.Marquee.start();
        }, 10000);
    }, 5000);
  }
}
</script>

<style lang="less" scoped>
.container {
  display: flex;

  .box {
    width: 200px;
    padding: 30px 10px;
    margin-right: 50px;
    background: #f5f5f5;
  }
}
</style>

# API

# 属性

参数说明类型可选值默认值
offset内容之间的间距Number—0
delay滚动前的延时Number—0
loop循环次数，默认无限循环Number—-1
speed速度，定时器循环时间，单位ms，每次移动1pxNumber—50
auto是否自动开始滚动Boolean—true
center内容宽度不需要滚动时是否居中Boolean—true

# 方法

方法名说明参数
start开始滚动，如果调用了stop方法，start会在之前停的地方继续滚动
stop暂停滚动
reset停止滚动且恢复到起始位置
