# ProcessStepList 步骤列表组件

一般用于带阶段性处置的事件的步骤信息显示及处理控件显示。

# 安装

$ npm i processsteplist

# 引入

import ProcessStepList from 'processsteplist'
Vue.use(ProcessStepList)

# 基础用法

内置了MediaViewList组件，当为文件类型的时候通过该组件进行显示，字段详见该组件文档。

        # 带操作控件

        • 事件信息
          事件类型机动车违停
          发生时间2020-07-11 10:10:10
        • 网格员处理
          主标题
          处理结果
          处理意见
        用法
        <template>
    <div class="container">
        <ProcessStepList
            :data="stepData2"
            :doingIcon="{color: '#2196F3'}"
            :showCollapse="true"
            style="padding: 16px 16px;padding-right: 20px;"
        >
            <template #operation="{item, index}">
                <!-- 事件处理控件 -->
                <StepOperationItem mainTitle="主标题" title="处理结果" v-if="item.operations.includes('handleSelect')">
                    <el-radio-group
                        v-model="item.operationData.handleType"
                        type="simple"
                    >
                        <el-radio-button :label="1">执法处理</el-radio-button>
                        <el-radio-button :label="2">不处理</el-radio-button>
                    </el-radio-group>
                </StepOperationItem>
                <!-- 处理意见输入控件 -->
                <StepOperationItem title="处理意见" v-if="item.operations.includes('opinionInput')">
                    <el-input
                        class="resize-area"
                        type="textarea"
                        :rows="4"
                        resize
                        :maxlength="128"
                        v-model="item.operationData.suggestion"
                    ></el-input>
                </StepOperationItem>
                <!-- 提交控件 -->
                <StepOperationItem v-if="item.operations.includes('submitBtn')">
                    <el-button style="margin-top: 20px" type="primary">提交</el-button>
                </StepOperationItem>
            </template>
        </ProcessStepList>
    </div>
</template>

        # API

        # ProcessStepList 属性

        参数说明类型可选值默认值
        data要显示的数据，数组类型，数组内每一项为对象，对象字段详见表1-1Array—[]
        doingIcondoing类型的icon的颜色配置Object—{color: '#02BF0F'}
        doneIcondone类型的icon的颜色配置Object—{color: '#02BF0F'}
        normalIconnormal类型的icon的颜色配置Object—{color: '#02BF0F'}
        showCollapse是否显示展开收缩按钮，全局控制，如果要在每个步骤里单独控制，请通过步骤数据里的hideCollapse字段控制Boolean—false
        dir信息的名称和内容的排列方式Stringv重置排列、h水平排列h
        padding是否需要内边距，开启后默认会加上padding: 16px 18px;的内边距Boolean—false
        noValueShow透传给MediaViewList组件使用，详见该组件文档String—-
        simple极简模式，不显示左侧图标及竖线，v1.0.15+Boolean—false

        # 表1-1 ProcessStepList 数组字段data内的每一项字段结构

        字段说明类型可选值默认值
        title步骤标题String—
        tag步骤标题旁边的标签，对象详细字段格式见表1-4Object—
        icon步骤iconStringnormal（一点小圆点）、doing（三个点）、done（一个勾）
        showOperation是否显示该步骤的操作控件Boolean—false
        hideCollapse是否隐藏该步骤的展开收缩按钮Boolean—
        collapse是否默认收起Boolean—false
        items要显示的信息列表，数组类型，数组内每一项为对象，对象字段结构详见表1-2Array—

        # 表1-2 ProcessStepList data字段内的数组字段items内的每一项字段结构

        字段说明类型可选值默认值
        name信息标题String—
        value信息内容，文字信息为String类型，文件（图片、视频、音频）列表类型为ArrayString/Array—
        file是否是文件列表类型Boolean—
        icon标题文字右边显示的icon，字段结构详见表1-3Object—
        custom使用自定义组件展示该条信息，自定义组件通过slot传进来，slot名为：showBoolean—

        # 表1-3 ProcessStepList data字段内的items字段内的icon字段结构

        字段说明类型可选值默认值
        show是否显示iconBoolean—
        iconicon类名String—
        titleDOM元素的title属性值String—
        handleicon点击的处理函数Function—

        # 表1-4 ProcessStepList data字段内的items字段内的tag字段结构

        字段说明类型可选值默认值
        type标签类型，内部使用的是hui的el-tag组件，详细文档：http://hui-vue.hikvision.com.cn/zh/data/tag.html#%E5%9F%BA%E7%A1%80%E7%94%A8%E6%B3%95Booleanprimary/gray/success/warning/danger
        value标签内容String—

        # StepOperationItem 属性

        参数说明类型可选值默认值
        mainTitle大标题String—
        title正常标题String—
        required是否显示必填标志，显示在正常标题右侧Boolean—false

        # ProcessStepList slot

        slot名说明参数
        operation操作控件slot，所有的操作控件都需要写在这里解构获取{item, index}，分别为该项的数据和索引
        show当内置的组件满足不了你的数据显示，你可以将自定义的展示组件写在这里解构获取{title, item, index}，分别为该项的标题、数据和索引

        # 事件

        事件说明回调函数
        collapse折叠事件index（触发重叠的流程索引）
