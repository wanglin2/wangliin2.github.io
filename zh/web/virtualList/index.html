# VirtualList 虚拟列表组件

页面DOM数量越多页面性能越低，所以对于超长列表来说不可能把全部节点都展示出来，虚拟列表就是只创建当前可视部分的节点，根据滚动到的位置动态替换节点的数据，以达到用有限节点显示无限数据的效果。

该组件只支持每项高度一样的情况。

# 安装

$ npm i v-list

# 引入

import VirtualList from 'v-list'
Vue.use(VirtualList)

# 基础用法

基础用法
<template>
  <div class="container">
    <VirtualList ref="virtualList" :itemSize="34" :step="34*2">
        <template #default="{item}">
            <div class="item">
                <span class="time">{{item.date}}</span>
                <span class="value" :title="item.eventType">{{item.eventType}}</span>
                <span class="tag" :title="item.status">{{item.status}}</span>
            </div>
        </template>
    </VirtualList>
  </div>
</template>

<script>
const random = (arr) => {
  return arr[Math.floor(Math.random() * arr.length)]
}
let list = []
for (let i = 0; i < 100; i++) {
    list.push({
        id: i,
        date: '2020-07-13',
        eventType: random(['出店经营', '垃圾堆积', '沿街晾晒', '违规宣传物', '其他']) + i,
        status: random(['已处理', '处理中', '未处理', '已超时'])
    })
}
export default {
  data () {
    return {}
  },
  mounted () {
    setTimeout(() => {
        this.$refs.virtualList.addData(list)
    }, 1000)
  }
}
</script>

# API

# 属性

参数说明类型可选值默认值
itemSize每一项内容的高度，单位pxNumber—
startOffset触发到顶事件的偏移量Number—0
endOffset触发到底事件的偏移量Number—0
step手动滚动步长，用于scrollUp和scrollDown方法Number—每项的高度itemSize的值

# 事件

事件说明回调函数
scrollTop滚动到顶部
scrollBottomscrollBottom

# 方法

方法名说明参数
addData添加数据，数组类型，每项为对象，必须要有不能重复的id字段data
scrollUp向上滚，滚动步长默认为itemSize的值，可通过step属性进行设置
scrollDown向下滚，同scrollUp

# slot

名称说明作用域插槽
default每一项内容的模板解构方式获取，字段item，该项的数据
