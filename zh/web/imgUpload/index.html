# ImgUpload 图片选择上传组件

# 安装

$ npm i img-upload

# 引入

import ImgUpload from 'img-upload'
Vue.use(ImgUpload)

# 基础用法

基础用法
<template>
  <div class="container">
    <ImgUpload v-model="fileList" :col="4"></ImgUpload>
  </div>
</template>

<script>
export default {
  data () {
    return {
        fileList: []
    }
  }
}
</script>

<style lang="less" scoped>

</style>

# 初始图片

用法
<template>
  <div class="container">
    <ImgUpload v-model="fileList2" :col="4"></ImgUpload>
  </div>
</template>

<script>
export default {
  data () {
    return {
        fileList2: [require('../../../.vuepress/public/1.jpg'), require('../../../.vuepress/public/2.jpg')]
    }
  }
}
</script>

# 属性

参数说明类型可选值默认值
fileList组件绑定的v-model，文件对象数组Array—[]
max可以选择的最大文件数量Number—
size文件大小限制，单位：MNumber—
correct是否开启角度矫正，开启此配置后返回的文件类型会变成blob，base64格式Boolean—false
compressWidth开启压缩，压缩后的宽度，高度根据原比例进行计算，开启此配置后返回的文件类型会变成base64格式Number—
compressHeight开启压缩，压缩后的高度，宽度根据原比例进行计算，开启此配置后返回的文件类型会变成base64格式Number—
col图片列数Number—3
margin每项间距，单位：pxNumber—10
