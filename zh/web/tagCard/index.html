# 数据的增删

关联场所1/2
  • 姗娜娜美容店
    出店经营垃圾堆积沿街晾晒违规宣传物
  • 家家乐超市
    出店经营垃圾堆积沿街晾晒违规宣传物
用法
<template>
  <div class="container">
    <TagCard 
        title="关联场所" 
        v-model="value2" 
        :enableCheck="true"
        :autoGoFirstWhenAdd="true"
    >
        <template #titleRight="{data:delData, index: delIndex}">
            <i class="h-icon-delete deleteBtn" @click="delLink(delData, delIndex)"></i>
        </template>
        <el-button class="addLinkBtn" icon="h-icon-add" type="primary" @click="addLinkPlace">关联场所</el-button>
    </TagCard>
  </div>
</template>

<script>
export default {
  data () {
    return {
      value2: ['姗娜娜美容店', '家家乐超市'].map(item => {
        return {
            title: item,
            tags: [
            {
                id: '1',
                name: '出店经营',
                checked: true
            },
            {
                id: '2',
                name: '垃圾堆积',
                checked: false
            },
            {
                id: '3',
                name: '沿街晾晒',
                checked: true
            },
            {
                id: '4',
                name: '违规宣传物',
                checked: false
            }
            ]
        }
      })
    }
  },
  methods: {
      /**
     * @Author: 王林25
     * @Date: 2020-07-20 15:54:29
     * @Desc: 删除关联场所
     */
    delLink (data, index) {
      this.value2.splice(index, 1)
    },

    /**
     * @Author: 王林25
     * @Date: 2020-07-20 16:27:10
     * @Desc: 添加关联场所
     */
    addLinkPlace () {
      this.value2.unshift({
        title: '黄焖鸡米饭' + Math.floor(Math.random() * 10),
        tags: [
          {
            id: '1',
            name: '出店经营',
            checked: false
          },
          {
            id: '2',
            name: '垃圾堆积',
            checked: false
          },
          {
            id: '3',
            name: '沿街晾晒',
            checked: false
          },
          {
            id: '4',
            name: '违规宣传物',
            checked: false
          }
        ]
      })
    }
  }
}
</script>

# API

# 属性

参数说明类型可选值默认值
title标题String—
data组件v-model绑定的数据，对象数组，对象格式见表1-1Array—[]
noDataShow没有数据时显示的字符String—-
dir没有数据时的排列方式Stringv重置排列、h水平排列h
showPageInfo是否显示页数信息Boolean—true
enableDragChange是否允许拖动切换Boolean—true
threshold拖动切换阈值，拖动的距离超过宽度的多少比例就切换到下一张Number0-1之间的小数0.5
enableLimitDrag是否允许当前第一项时继续向右拖动、当前最后一项时继续向左拖动Boolean—true
enableSpeedChange是否允许快速滑动切换Boolean—true
speedThreshold快速切换的速度阈值，px/sNumber—800
enableCheck开启标签选择Boolean—false
autoGoFirstWhenAdd增加数据时自动切换回第一个位置Boolean—false

# 表1-1 data字段格式

字段名称说明类型可选值默认值
title标题String—
tags标签列表，对象数组，对象格式见表1-1-1Array—[]

# 表1-1-1 tags字段格式

字段名称说明类型可选值默认值
id标签id，需确保唯一Number/String—
name标签名称String—
checked是否选中Boolean—

# 事件

事件说明回调函数
currentChange页数切换事件item（切换到的这项数据）、curIndex（切换后的索引）、lastIndex（切换前的索引）
tagChecked标签选中与取消选中事件item（当前这项标签数据）、index（标签所在的该项索引）, tagIndex（标签索引）

# 方法

方法名说明参数
deleteItem删除某项数据index（要删除的索引）
go切换到指定索引项index（要切换到的索引）
goLeft向左切换
goRight向右切换
