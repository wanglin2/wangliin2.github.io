# API

# 属性

参数说明类型可选值默认值
title标题，只有在无数据的情况下才会显示String—
tip提示信息，不为空的话顶部会显示一条提示信息String—
data数据，对象数组，对象格式见表1-1Array—[]
noDataShow没有数据时的显示字符String—-
dir排列方式Stringv重置排列、h水平排列h
showBtnText是否显示上一条下一条按钮文案，设为false只显示箭头Boolean—true
prevText向前翻页按钮文字String—上一个
nextText向后翻页按钮文字String—下一个
