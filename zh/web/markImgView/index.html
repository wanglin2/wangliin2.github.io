# MarkImgView 图片预览组件

和MarkerImg类似，但是功能更全，且支持多图。

# 安装

$ npm i markimgview

提示

如果报Cannot find module '@hui-pro/img-preview'，current-script-polyfill、@hui-pro/img-carousel错误，请手动在当前项目安装一下这些依赖。

# 引入

import MarkImgView from 'markimgview'
Vue.use(MarkImgView)

# 基础用法

需要自己覆盖一下浏览器默认样式。

基础用法
<template>
    <div class="container">
        <MarkImgView :src="src" :mark="mark" :showTopToolBar="false" :showBottomToolBar="true" :heightOffset="heightOffset"></MarkImgView>
    </div>
</template>

<script>
export default {
  data () {
    return {
      src: require('../../../.vuepress/public/94fb997b-f094-473a-b5fd-c0d3520f742f.png'),
      mark: [
            {
                ratio: false,
                xmin: 278,
                xmax: 278+313,
                ymin: 191,
                ymax: 191+357,
                text: '我是文字',
                color: '#FA3239',
                fillColor: 'rgba(250,50,57,0.30)',
                fontColor: '#fff'
            }
        ],
        heightOffset: 0
    }
  },
  mounted () {
    this.heightOffset = window.innerHeight - 500
  }
}
</script>

<style lang="less" scoped>
.container {
    width: 100%;
    height: 500px;

        list-style: none;
    }

    /deep/ button {
        outline: none;
        border: none;
    }

    /deep/ * {
        margin: 0;
        padding: 0;
    }
}

</style>

}
</style>

# API

# 属性

参数说明类型可选值默认值
heightOffset组件高度默认为浏览器可视区域高度-heightOffset，比如你需要的高度是500px，则该值传浏览器可视区域高度-500。Number—0
src图片url，字符串或数组，数组的话每项是图片。String/Array—
mark图片上标注框及文字信息Array—[]
fullScreenCanDrag全屏显示下是否可拖动Boolean—true
fullScreenCanScale全屏显示下是否可缩放Boolean—true
showTopToolBar是否显示顶部工具栏Boolean—true
showBottomToolBar是否显示底部工具栏Boolean—false
showAlbum是否显示相册，仅在src为数组形式下Boolean—false
scaleDrag是否图片放大后才可拖动Boolean—false
downloadFn自定义下载函数，回调参数为src的值Function—
downloadUrl自定义下载urlString/Array—
noImg没有图片时的默认图String—
failImg图片加载失败时的默认图String—
blurBg是否显示虚化背景Boolean—默认为纯黑背景
blurNum虚化度Number—15

# 方法

方法名说明参数
reInit重新初始化
download下载图片
zoomIn放大
zoomOut缩小
