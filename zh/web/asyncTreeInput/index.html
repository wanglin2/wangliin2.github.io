# AsyncTreeInput 异步树输入框组件

基于hui-pro异步树组件SyncTree封装，SyncTree组件文档：http://emptycup.hikvision.com.cn/hui-pro/zh/widget/data/sync-tree.html。

# 安装

$ npm i asynctreeinput

提示

如果报错@hui-pro/sync-tree未找到，可以在当前项目安装该组件。

# 引入

import AsyncTreeInput from 'asynctreeinput'
Vue.use(AsyncTreeInput)

# API

# 属性

参数说明类型可选值默认值
placeholder输入框占位字符String—请选择
searchPlaceholder搜索输入框占位字符String—请搜索
getTreeData异步获取下一个节点数据的函数，回调参数依次为parentId（要请求子节点的节点id）, data（当前节点数据）, keyWord（搜索关键词，如果该字段存在，那么parentId、data都为null）Function—
onlyShowWhen该函数用来判断当前点击的这个节点是否可以选中，回调参数为data、node，依次为当前点击节点的数据和节点对象—
treeMap树属性配置Object—{label: 'name',isLeaf: 'leaf',icon: 'icon'}

# 事件

事件说明回调函数
treeItemSelecteChange树节点选择事件回调参数为data、node，依次为当前点击节点的数据和节点对象
