# thDropdownMixin 表头下拉菜单混入

hui表格表头不支持slot，所以常见的点击icon显示下拉菜单或其他信息没有现成方案，本mixin可用来帮助解决该问题。

# 安装

$ npm i th-dropdown-mixin

# 引入

import thDropdownMixin from 'th-dropdown-mixin'
export default {
  mixins: [thDropdownMixin]
}

# 基础用法

点击状态一栏表头上的过滤icon会显示下拉菜单。

使用步骤：

1.引入mixin。

2.通过html方式在hui表头上插入你的DOM结构，给icon元素加上id。

3.随便在哪定义你的下拉菜单或其他要显示的元素，并给最外层元素加上动态样式属性：:style="{...dropDownInitStyle,...dropDownStyle}"。

4.在mounted里调用初始化方法this.initDropDown(icon元素id或icon dom元素)。

5.搞定。

可参考下方示例代码。

暂无数据
  • 全部
  • 待审核
  • 待审批
  • 建设中
基础用法
<template>
    <div class="container">
        <el-table
          :data="tableData"
          show-overflow-title
          force-scroll="horizontal"
          stripe
        >
          <el-table-column type="selection"></el-table-column>
          <el-table-column prop="address" label="增补地址" />
          <el-table-column prop="region" label="所属辖区"></el-table-column>
          <el-table-column prop="reportTime" label="上报时间" />
          <el-table-column prop="sourceType" label="建议来源"></el-table-column>
          <el-table-column
            prop="status"
            label="<div class='filterBox'>状态<i class='filterIcon h-icon-filter' id='filterIcon' /></div>"
            :ishtml="true"
          ></el-table-column>
          <el-table-column label="操作" width="150">
            <template slot-scope="scope">
              <el-button
                type="iconButton"
                size="mini"
                icon="h-icon-access"
                @click="toDetail(scope)"
                title="处理"
              />
            </template>
          </el-table-column>
        </el-table>
        <ul
            class="el-dropdown-menu el-popper"
            :style="{...dropDownInitStyle,...dropDownStyle}"
        >
            <li class="el-dropdown-menu__item" v-for="item in pendingStatusList" :key="item.value">{{item.name}}</li>
        </ul>
    </div>
</template>

<script>
import thDropdownMixin from 'th-dropdown-mixin'
export default {
  mixins: [thDropdownMixin],
  data () {
    return {
        tableData: [],
        pendingStatusList: [
            { name: '全部', value: '' },
            { name: '待审核', value: 1 },
            { name: '待审批', value: 2 },
            { name: '建设中', value: 3 }
        ]
    }
  },
  mounted () {
    this.tableData = new Array(10).fill(0).map((item, index) => {
        return {
            id: index,
            address: '拱墅区古运路142号附近' + index,
            region: '塘河新村/第一网格',
            reportTime: '2020/03/21 14:24:51',
            sourceType: '人工上报',
            status: '待审核'
        }
    })
    this.initDropDown('#filterIcon')
  }
}
</script>

<style lang="less" scoped>
.container {
    /deep/ table {
        margin: 0
    }

    /deep/ .filterBox {
        display: flex;

        .filterIcon {
            font-size: 20px;
            margin-left: 5px;
            cursor: pointer;
        }
    }
}
</style>
