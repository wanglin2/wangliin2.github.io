# QuickInput 快速输入框

依赖于vant的Field组件，文档地址：https://youzan.github.io/vant/#/zh-CN/field。

# 安装

$ npm i quick-input

# 引入

import QuickInput from 'quick-input'
Vue.use(QuickInput)

# 基础用法

  • 车辆已开走。
  • 车主无响应，车辆未开走。
  • 未发生违规事件。
  • 不归本辖区管理。
基础用法
<template>
  <div class="container">
    <QuickInput 
      v-model="suggestion" 
      placeholder="请输入处理意见"
      :list="suggestionList"
    ></QuickInput>
  </div>
</template>

<script>
export default {
  data () {
    return {
      suggestion: '',
      suggestionList: [
          '车辆已开走。',
          '车主无响应，车辆未开走。',
          '未发生违规事件。',
          '不归本辖区管理。'
      ]
    }
  }
}
</script>

<style lang="less" scoped>
.container {
  width: 375px;
  height: 667px;
  background-color: #f5f5f5;
  padding: 15px;
}
</style>

# 带错误提示

请输入处理意见
  • 车辆已开走。
  • 车主无响应，车辆未开走。
  • 未发生违规事件。
  • 不归本辖区管理。
用法
<template>
  <div class="container">
    <QuickInput 
      v-model="suggestion" 
      placeholder="请输入处理意见"
      :list="suggestionList"
      :errorMessage.sync="errorMessage"
    ></QuickInput>
  </div>
</template>

<script>
export default {
  data () {
    return {
      suggestion: '',
      suggestionList: [
          '车辆已开走。',
          '车主无响应，车辆未开走。',
          '未发生违规事件。',
          '不归本辖区管理。'
      ],
      errorMessage: '请输入处理意见'
    }
  }
}
</script>

# API

# 属性

参数说明类型可选值默认值
value组件绑定的v-model值Number/String—
list快速输入列表，纯字符串数组Array—[]
errorMessage错误提示信息，可带.sync修饰符，这样当输入框非空时会自动清除提示，如果不需要自动清除的话则不用带String/Number—
placeholder输入框占位提示文字String—
maxlength输入的最大字符数Number—
autosize是否自适应内容高度Boolean—false
props传入其他vant属性，对象类型Object—{}
