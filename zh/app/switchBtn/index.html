# SwitchBtn 按钮组

# 安装

$ npm i switch-btn

# 引入

import SwitchBtn from 'switch-btn'
Vue.use(SwitchBtn)

# 基础用法

已完成
已处理
未处理
已违规
误报
退回
已整改
基础用法
<template>
  <div class="container">
    <div class="item">
      <SwitchBtn :data="auditType"></SwitchBtn>
    </div>
    <div class="item">
      <SwitchBtn :data="auditType2"></SwitchBtn>
    </div>
    <div class="item">
      <SwitchBtn :data="auditType3"></SwitchBtn>
    </div>
  </div>
</template>

<script>
export default {
  data () {
    return {
      auditType: [
          { text: '已完成', value: 1 }
      ],
      auditType2: [
          { text: '已处理', value: 1 },
          { text: '未处理', value: 2 }
      ],
      auditType3: [
          { text: '已违规', value: 1 },
          { text: '误报', value: 2 },
          { text: '退回', value: 4 },
          { text: '已整改', value: 3 }
      ],
    }
  }
}
</script>

}

# API

# 属性

参数说明类型可选值默认值
data按钮数据，数组类型，每一项为对象，对象格式详见表1-1Array—[]
initValue指定初始选中值Number/String—
small是否是小型模式Boolean—false
color自定义颜色值String—#3A93FF

# data字段格式

参数说明类型可选值默认值
text按钮的文字Number/String—
value按钮的值Number/String—

# 事件

事件说明回调函数
change按钮切换value（切换到按钮的值）、data（切换到按钮的数据）
