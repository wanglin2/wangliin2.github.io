# Observer 发布订阅库

# 安装

$ npm i observer

# 引入

import Observer from 'observer'
let observer = new Observer(opt)

# API

# 实例方法

方法名说明参数
publish发布话题topic（话题名称）, ...arg（要传递的数据）
subscribe订阅某个话题，会返回一个token，用于取消订阅topic（话题名称）, fn（回调函数）
unsubscribe解除订阅token（订阅话题时返回的token）
clearTopic删掉某一类话题及其订阅topic（话题名称）
clearAll删除所有话题及订阅
